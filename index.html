About

I build complete 3D browser games in pure Three.js / WebGL — no Unity, no engine. My focus is performance and real game systems: economy, progression and AI. I also handle integration with game distribution platforms such as the CrazyGames SDK, shipping polished titles that run instantly in any modern browser.

Gunsmith Range

A 3D FPS campaign built in pure Three.js — 10 missions, a weapon system with attachments, AI enemies and CrazyGames SDK integration.

Three.js Vanilla JS · ES Modules WebGL Web Audio API CrazyGames SDK v2
  • 10 campaign missions across unique locations (warehouse, outpost, street, house, industrial site, bunker, airport, train, port, complex)
  • 8 weapons (rifle, pistol, SMG, DMR, shotgun, LMG + 2 purchasable: OSA bullpup, GRZMOT revolver) and ~14 attachments (sights, magazines, grips, suppressors)
  • Three AI enemy types (shooter, tactical crouch, knifeman) with procedural animation
  • Wave system with captain narration between stages
  • Economy (credits), weapon unlocks and progress saved in localStorage
  • Grenades (frag + flashbang) with flight physics
  • Touch controls on mobile (virtual joystick)
  • Optimized rendering (light culling, raycast filtering, GPU tuning)

  • Wardrobe of 11 unlockable character looks
